Política de Privacidade

DED+ Express · Atualizada em março de 2026
🔒 Extensão para o Chrome Web Store

Esta Política de Privacidade descreve como a extensão DED+ Express coleta, usa e protege as informações dos usuários. Ao utilizar a extensão, você concorda com os termos descritos neste documento.

1. Quem somos

O DED+ Express é uma extensão para o Google Chrome desenvolvida para auxiliar professores no cadastro automatizado de aulas no sistema DED da Secretaria de Educação do Estado de Minas Gerais (ded.educacao.mg.gov.br).

2. Dados coletados

A extensão coleta exclusivamente os seguintes dados:

  • Endereço de e-mail: utilizado para identificação e autenticação do professor no sistema. O e-mail é armazenado de forma segura no banco de dados Supabase e localmente no navegador via chrome.storage.local.
  • Contador de lançamentos: número de aulas cadastradas pelo professor, usado para controle do plano gratuito (limite de 10 lançamentos).
  • Status do plano: informação indicando se o professor adquiriu o plano Premium.
✅ Não coletamos senhas, dados bancários, histórico de navegação, localização, mensagens ou qualquer outro dado pessoal além do e-mail.

3. Como usamos os dados

Os dados coletados são usados exclusivamente para:

  • Autenticar o professor e manter a sessão ativa entre usos da extensão;
  • Controlar o número de lançamentos gratuitos disponíveis;
  • Verificar se o professor adquiriu o plano Premium após pagamento;
  • Enviar o link de acesso (magic link) para o e-mail informado.

4. Compartilhamento de dados

Os dados não são vendidos, alugados ou compartilhados com terceiros para fins comerciais. O e-mail e os dados de uso são armazenados no serviço Supabase (supabase.com), que opera em conformidade com as leis de proteção de dados aplicáveis.

O processamento de pagamentos é realizado pela plataforma Stripe (stripe.com), que recebe apenas o e-mail informado pelo professor para vinculação do pagamento. A extensão não tem acesso a dados de cartão de crédito ou informações bancárias.

5. Armazenamento e segurança

  • Os dados são armazenados de forma segura no banco de dados Supabase, com Row Level Security (RLS) habilitado;
  • O e-mail é salvo localmente no navegador via chrome.storage.local, acessível apenas pela própria extensão;
  • A comunicação com o servidor utiliza conexões criptografadas (HTTPS/TLS);
  • Nenhum dado é transmitido para servidores além dos descritos nesta política.

6. Serviços externos utilizados

  • Supabase (supabase.com) — banco de dados e autenticação;
  • Stripe (stripe.com) — processamento de pagamento do plano Premium;
  • Pollinations AI (text.pollinations.ai) — geração de títulos de aulas via inteligência artificial. Apenas o tema informado pelo professor é enviado; nenhum dado pessoal é transmitido.

7. Direitos do usuário

O professor tem o direito de:

  • Solicitar a exclusão de seus dados a qualquer momento;
  • Acessar quais dados estão armazenados em seu nome;
  • Revogar o acesso fazendo logout da extensão.

Para exercer esses direitos, entre em contato pelo e-mail indicado na seção de contato abaixo.

8. Retenção de dados

Os dados são mantidos enquanto o professor utilizar a extensão. Após solicitação de exclusão ou inatividade por mais de 12 meses, os dados poderão ser removidos permanentemente do banco de dados.

9. Cookies e rastreamento

A extensão não utiliza cookies de rastreamento, pixels de rastreamento, fingerprinting de dispositivo ou qualquer outra tecnologia de monitoramento comportamental.

10. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data da última atualização está indicada no topo deste documento. O uso continuado da extensão após alterações implica na aceitação da nova versão.
